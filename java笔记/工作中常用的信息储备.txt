启动远程桌面的方式与使用httpd

	1.在 [开始] 菜单中点击 [运行], 在输入框中输入 "mstsc /admin"(mstsc /admin:这里的中间空格不能少),表示用admin账户登录,如果输入的是 "mstsc" ,表示用普通账户登录

	2.工作常用的服务器转换器有 httpd ,但需要对其中的 httpd.conf 文件进行相应的配置,同时配置完了后需要进入相应的bin路径下是用 service httpd restart 命令进行重启

	  其实就是重启bin文件夹下的 httpd 文件.

java webservice 客户端常用的命令:

	实例1 : wsimport -s . http://192.168.1.100:6789/hello?wsdl

	实例2 : wsimport -s . -p com.wulin.ws.client http://192.168.1.100:6789/hello?wsdl

	解释: 1.wsimport 这其实是我们安装的jdk1.6.0.12之后的版本才带有一个与我们平常使用的

	Javac命令是同一个意思

	2. -s : 指定放置生成的源文件的位置

	3. . : 指的是当前路径

	4. -p 重新自定义包路径

	5. 路径 : 类似于 http://192.168.1.100:6789/hello?wsdl 这样的soap协议路径
